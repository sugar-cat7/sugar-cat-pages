# アクセシビリティチェックリスト

## 概要

WCAG 2.1の基準に基づいたアクセシビリティガイドラインです。

---

## 1. 代替テキスト

| チェック項目 | 詳細 |
|-------------|------|
| 画像に代替テキストが付与されている | すべての `<img>` タグに `alt` 属性を設定 |
| 装飾目的の画像が無視できるようになっている | 装飾画像には `alt=""` を設定 |

```tsx
// 情報を伝える画像
<img src="/article-image.jpg" alt="東京の夜景を上空から撮影した写真" />

// 装飾目的の画像
<img src="/decorative-line.png" alt="" />
```

---

## 2. マークアップ

| チェック項目 | 詳細 |
|-------------|------|
| 見出しが `<h1>〜<h6>` でマークアップされている | 見出しレベルを順序通りに使用 |
| リストが `<ul>`, `<ol>` でマークアップされている | 適切なリスト要素を使用 |
| 同じ `id` を持つ要素がページ内に複数存在しない | id属性は一意に |

```tsx
// ✅ OK: 順序通りの見出し
<h1>ページタイトル</h1>
<h2>セクション1</h2>
<h3>サブセクション1-1</h3>

// ❌ NG: 見出しレベルのスキップ
<h1>ページタイトル</h1>
<h3>セクション1</h3>  {/* h2をスキップ */}
```

---

## 3. 見やすさ

| チェック項目 | 詳細 |
|-------------|------|
| 背景色と文字色のコントラスト比が4.5:1以上 | 読みやすさの確保 |
| 色や形のみでコンテンツを説明していない | 複数の手段で情報を伝える |

### コントラスト比の基準

| 対象 | 最小コントラスト比 |
|------|-------------------|
| 通常テキスト | 4.5:1 |
| 大きなテキスト（18px以上） | 3:1 |

---

## 4. 操作しやすさ

| チェック項目 | 詳細 |
|-------------|------|
| キーボードで操作可能 | インタラクティブ要素にフォーカスできる |
| キーボード操作の順序が見た目の順序とあっている | 自然なtab順序 |

```css
:focus-visible {
  outline: 2px solid var(--color-border);
  outline-offset: 2px;
}
```

---

## 5. ナビゲーション

| チェック項目 | 詳細 |
|-------------|------|
| ページの言語が `<html>` に記載されている | `<html lang="ja">` |
| ページのタイトルがページの内容を表している | 具体的で一意なタイトル |
| リンクのテキストからリンク先が判別できる | 「こちら」のみのリンクは避ける |

```tsx
// ❌ NG: リンク先が不明確
<a href="/policy">こちら</a>をご確認ください。

// ✅ OK: リンク先が明確
<a href="/policy">プライバシーポリシー</a>をご確認ください。
```
