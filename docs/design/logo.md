# ロゴガイドライン

## 概要

ロゴは、ブランドのアイデンティティを表現する重要な要素です。使用する領域や媒体に応じて、適切な形式とガイドラインに従って運用してください。

## 領域別使用ガイドライン

### 映像での使用

映像コンテンツでロゴを使用する場合は、以下のガイドラインに従ってください。

| 項目 | 推奨事項 |
|------|----------|
| 表示時間 | 最低2秒以上の表示を確保する |
| 位置 | 画面の端から十分なマージンを取る |
| サイズ | 視認性を確保できる最小サイズ以上で使用する |
| アニメーション | ブランドイメージを損なわない控えめな動きに留める |
| 背景 | コントラストを確保し、ロゴが明瞭に見える背景色を選択する |

#### 映像フォーマット別の注意点

- **動画広告**: 冒頭または末尾でのロゴ表示を推奨
- **プロダクト紹介動画**: ウォーターマークとしての使用は控えめに
- **SNS用動画**: 正方形・縦型フォーマットに合わせたロゴ配置を検討

### サウンドでの使用

音声コンテンツやサウンドロゴを使用する場合は、以下のガイドラインに従ってください。

| 項目 | 推奨事項 |
|------|----------|
| 長さ | 3〜5秒程度のサウンドロゴを推奨 |
| 音量 | コンテンツ全体と調和する音量レベルに調整 |
| 使用タイミング | コンテンツの開始時または終了時に配置 |
| トーン | ブランドイメージに合致した音色・メロディを使用 |

#### サウンド使用シーン

- **ポッドキャスト**: イントロ・アウトロでの使用
- **動画コンテンツ**: 映像ロゴと組み合わせて使用
- **音声広告**: 認知向上のためのサウンドブランディング

## 各領域での注意事項

### 禁止事項

- ロゴの変形・歪曲
- 指定外のカラーでの使用
- 低解像度での使用
- 背景とのコントラスト不足による視認性の低下
- 他の要素との過度な近接配置

### 推奨事項

- 常にオリジナルのロゴデータを使用する
- 適切なクリアスペースを確保する
- 使用目的に応じた適切なファイル形式を選択する
  - Web: SVG または PNG
  - 印刷: AI、EPS、または高解像度PDF
  - 映像: 透過PNGまたはアルファチャンネル付き動画形式
