# ライティングガイドライン

## 概要

伝わる文章を書くことは、ユーザー体験を向上させる重要な要素です。

---

## 3つの基本原則

### 1. 目的に合わせた情報の取捨選択

読み手の状況を踏まえ、言葉にする内容や表現を厳選します。

### 2. 適切な言葉選び

- 肯定的な表現を優先する
- 曖昧な表現を避ける
- 読者に馴染みのある言葉を使う

### 3. 正しい文法

- 主語と述語の対応
- 修飾語の位置
- 助詞の適切な使用

---

## 実践ルール

### 一文は50文字程度

```
❌ NG: この機能を使用することで、ユーザーは自分のアカウント設定画面から
      通知の受信設定を変更することができるようになります。

✅ OK: この機能でアカウント設定から通知設定を変更できます。
```

### 主語と格助詞の明確化

```
❌ NG: データ削除しました。
✅ OK: データを削除しました。
```

### 二重否定を避ける

```
❌ NG: 入力しないと完了できません。
✅ OK: 入力すると完了できます。
```

### 表記の統一

| カテゴリ | ルール |
|----------|--------|
| 漢字 | 常用漢字を基本とする |
| 数字 | 半角数字、3桁ごとにカンマ |
| 記号 | 全角を使用（句読点、括弧など） |
