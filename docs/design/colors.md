# カラーガイドライン

## 概要

カラーパレットは、ブランドの視覚的アイデンティティを構成する重要な要素です。一貫したカラー使用により、ユーザーに統一感のある体験を提供します。

## ブランドカラー

ブランドを代表する主要なカラーです。

| 名称 | HEX | RGB | 用途 |
|------|-----|-----|------|
| オフホワイト | `#F7F6F3` | rgb(247, 246, 243) | 背景・ベースカラー |
| ライムイエロー | `#E6EC49` | rgb(230, 236, 73) | アクセント・強調 |

### オフホワイト（Background）

メインの背景色として使用します。温かみのあるオフホワイトにより、柔らかく親しみやすい印象を与えます。

```css
--color-background: #F7F6F3;
```

### ライムイエロー（Accent）

CTAボタンや重要な要素の強調に使用します。視認性が高く、ユーザーの注意を引きつけます。

```css
--color-accent: #E6EC49;
```

## セカンダリカラー

ブランドカラーを補完し、UIのバリエーションを豊かにするカラーです。

| 名称 | HEX | RGB | 用途 |
|------|-----|-----|------|
| ウォームベージュ | `#CBC6BE` | rgb(203, 198, 190) | フッター・セカンダリ背景 |
| ミントグリーン | `#97D5D9` | rgb(151, 213, 217) | シアン系・補助アクセント |
| ビビッドオレンジ | `#F7643D` | rgb(247, 100, 61) | 警告・注目要素 |
| ソフトブルー | `#83A8F9` | rgb(131, 168, 249) | 情報・リンク |
| ライトグレー | `#D9D9D9` | rgb(217, 217, 217) | 装飾・ボーダー |

### 使用例

```css
/* フッター背景 */
.footer {
  background-color: #CBC6BE;
}

/* 情報バッジ */
.badge-info {
  background-color: #83A8F9;
}

/* 警告メッセージ */
.alert-warning {
  border-color: #F7643D;
}
```

## カラー使用ルール

### 優先順位

1. **背景**: オフホワイト `#F7F6F3` を基本とする
2. **アクセント**: ライムイエロー `#E6EC49` を主要なCTAに使用
3. **セカンダリ**: 用途に応じてセカンダリカラーを選択

### コントラスト比の考慮

アクセシビリティを確保するため、テキストと背景のコントラスト比は以下を満たす必要があります：

| 対象 | 最小コントラスト比 |
|------|-------------------|
| 通常テキスト（14px未満） | 4.5:1 以上 |
| 大きなテキスト（18px以上、または14px以上のボールド） | 3:1 以上 |
| UIコンポーネント・グラフィック | 3:1 以上 |

### カラーの組み合わせ

#### 推奨される組み合わせ

| 背景 | テキスト/前景 | 用途 |
|------|--------------|------|
| オフホワイト | ダークグレー/ブラック | 本文テキスト |
| ライムイエロー | ダークテキスト | CTAボタン |
| ウォームベージュ | ダークテキスト | フッターセクション |

#### 避けるべき組み合わせ

- ライムイエロー背景に白テキスト（コントラスト不足）
- ミントグリーン背景にソフトブルーテキスト（色の区別が困難）

## 禁止事項

- パレット外のカラーの無断使用
- ブランドカラーの変形（透明度の過度な変更、グラデーションへの変換等）
- コントラスト比を満たさない組み合わせでのテキスト表示
- 色のみで情報を伝えること（形状やテキストも併用する）
